Babel is JavaScript Compiler

link BabelJS

1. Install Babel
1-1. Install babel node : install @babel/node 1-2. Install babel preset-env : npm install @babel/preset-env 1-3. Install babel core : npm install @babel/core

2. Create file ".babelrc" and write Preset
    {
        "presets": [
            [
            "@babel/preset-env",
            {
                "useBuiltIns": "entry"
            }
            ]
        ]
    }
3. Edit file "package.json"
    ...

    "scripts": {
        "start": "babel-node index.js"
    },

    ...
Etc. Install "nodemon"
If when you modify source code and save, auto-restart server
1. Install nodemon : npm install nodemon -D 
        "devDependencies": {
            "nodemon": "^1.18.11"
        }
2. Edit file package.json 
        ...

        "scripts": {
            "start": "nodemon --exec babel-node index.js --delay 2" // restart server after 2 sec 
        },

        ...
