Pug

: View-engine on Express!
Link

How to
Install : npm install pug Import : import pug from 'pug' Use
1. Create folder "views" : Template folder default name is 'views' on ExpressJs. 2. Set "Pug" on Express app : app.set('view_engine', pug); 3. Create pug file : Create file. extended name is ".pug".
